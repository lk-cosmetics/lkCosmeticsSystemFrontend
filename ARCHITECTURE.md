## ğŸ¯ Key Integration Points

### 1. Login Flow

- User submits form â†’ LoginForm calls `authStore.login()`
- AuthStore calls `authService.login()`
- Service makes API call, stores tokens
- Store updates state, triggers re-render
- Navigate to dashboard

### 2. API Calls

- Component calls `apiClient.get()`
- Request interceptor adds Bearer token
- If 401 error, response interceptor refreshes token
- Original request retried automatically

### 3. Route Protection

- User navigates to protected route
- ProtectedRoute checks `authStore.isAuthenticated`
- If true, render route; if false, redirect to login

### 4. Logout

- User clicks LogoutButton
- Calls `authStore.logout()`
- Clears tokens and user data
- Navigates to login page
